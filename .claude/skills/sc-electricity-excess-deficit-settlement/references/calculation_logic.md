## 计算公式详解

### Step 1：计算分时加权均价（通用函数）

```python
def weighted_avg_price(qty, price):
    """分时加权均价 = Σ(qty × price) / Σ(qty)"""
    fee = qty * price   # (days, 24)
    return fee.sum(axis=0) / qty.sum(axis=0)  # (24,)

# 实时分时加权均价 = weighted_avg_price(实时市场出清电量, 实时结算电价)
# 月内集中交易分时均价 = weighted_avg_price(集中交易电量, 集中交易价格)
```

### Step 2：各时段汇总

```python
时段实际用电汇总[t] = Σ(d=1..31) 实际用电[d][t]
时段合约电量汇总[t] = Σ(d=1..31) 合约电量[d][t]
```

### Step 3：三段式判断（互斥，同一时段只触发一种）

```python
ratio = 合约汇总 / 实际汇总   # 比率

if 合约汇总 < 实际汇总 × 0.85:
    # 缺额偏差：ΔP = max(0, 集中价 − 实时价)
    deficit = 实际汇总 × 0.85 − 合约汇总
    recovery = 1.1 × max(0, 月内集中均价[t] − 实时均价[t]) × deficit
elif 合约汇总 > 实际汇总 × 1.15:
    # 超额偏差：ΔP = max(0, 实时价 − 集中价)
    excess = 合约汇总 − 实际汇总 × 1.15
    recovery = 1.1 × max(0, 实时均价[t] − 月内集中均价[t]) × excess
else:
    recovery = 0.0   # 未触发
```

## 常见问题

**Q：价格差值为0或负值？**  
A：确认价格列是否正确读取。价格列在 cols 30-53（分时排列），不是 cols 28-29（日均价列）。

**Q：所有时段均未触发？**  
A：说明合约与实际用电量拟合度好（偏差在±15%以内），符合实际业务情况。

**Q：缺额和超额同时不为0？**  
A：不可能。同一时段三选一互斥判断，每天各时段独立，不会同时触发两种条件。
